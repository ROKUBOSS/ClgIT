  • Basic-Theorem (মৌলিক-উপপাদ্য)

    • Addition
    A+0 = A

    A+1 = 1

    A+A = A

    A+ A = 1

    Multiplication

    A*0 = 0

    A*1 = A

    A*A = A

    A* A = 0

    Complement

    A (Double-Bar) = A

  • Cumulative-Theorem (বিনিময়-উপপাদ্য)

    • A+B = B+A

    A*B = B*A

  • Associative-Theorem (অনুষঙ্গ-উপপাদ্য)

    • A+(B+C) = (A+B)+C

    A(BC) = (AB)C

  • Distrubuted-Theorem (বিভাজন-উপপাদ্য)

    • A(B+C) = AB + AC

    A + BC = (A+B)(A+C)

  • Secondary-Theorem (সহায়ক-উপপাদ্য)

    • A+AB = A

    A(A+B) = A

    A+AB = A+B

    A+AB =A+B

  • Common নিয়ে সমাধান করতে হবে
  • এক পক্ষ কে অপর পক্ষর সহিত গুন করতে হবে

    • Simplification

    বামপক্ষ,
    A+AB
    = A(1 + B)
    = A*1
    = A
    = ডানপক্ষ
    অতএব A+AB = A (প্রমাণিত)

    বামপক্ষ,
    (A+B)(A+C)
    = AA + AC + AB + BC
    = A + AC + AB + BC
    = A(1 + C + B) + BC
    = A*1 + BC
    = A + BC
    = ডানপক্ষ
    অতএব (A+B)(A+C) = A+BC (প্রমাণিত)

    বামপক্ষ,
    A ( A + B)
    = AA + AB
    = 0 + AB
    = AB
    = ডানপক্ষ
    অতএব A ( A + B) = AB (প্রমাণিত)

    বামপক্ষ,
    (A + B)(A + B)
    = AA + AB + AB + BB
    =A + AB + AB + 0
    = A (1 + B + B)
    =A*1
    = A
    = ডানপক্ষ
    অতএব (A + B)(A + B) = A (প্রমাণিত)
